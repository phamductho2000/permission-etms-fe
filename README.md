# HCMA FRONTEND
## Install Guide

```bash
Step1: npm install or yarn
Step2: run backend (run devenv and run BE)
Step3: npm run codegen
```

### Start project

```bash
npm start
npm run start:dev
npm run start:test
npm run start:uat
npm run start:prod
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
